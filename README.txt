JEPARA ART FINISHING WEBSITE

============================
Website statis ini dibuat untuk Jepara Art Finishing, menampilkan testimoni hasil finishing ukiran kayu serta galeri produk.

ISI FILE:
---------
1. index.html     : Halaman utama website
2. sitemap.xml    : Sitemap untuk SEO
3. robots.txt     : Konfigurasi akses mesin pencari
4. favicon.ico    : (Silakan tambahkan sendiri file favicon Anda)
5. gambar1.jpg - gambar4.jpg : (Tambahkan gambar galeri dengan nama sesuai di HTML)

CARA PENGGUNAAN:
----------------
1. Unggah seluruh file ini ke root folder domain Anda di hosting/cPanel/FTP.
2. Pastikan file `favicon.ico` dan gambar-gambar berada di direktori yang sama.
3. Akses situs Anda di browser untuk memastikan tampilannya sesuai.
4. Untuk hasil maksimal, daftarkan website ke Google Search Console dan submit `sitemap.xml`.

KONTAK:
-------
Jika Anda butuh bantuan lanjut atau ingin menambahkan fitur lain,
silakan hubungi pembuat atau developer Anda.
